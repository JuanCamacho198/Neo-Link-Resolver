1️⃣  SPINNER GRANDE (size='lg')
    ├─ Color primario para detección
    ├─ Color positivo (verde) para resolución
    └─ Rotación continua mientras procesa

2️⃣  BARRA DE PROGRESO INDETERMINADA
    ├─ Animación fluida sin porcentaje
    ├─ Ancho completo del contenedor
    └─ Color acorde al paso (primario/positivo)

3️⃣  ETIQUETA DE ESTADO
    ├─ Se actualiza en tiempo real
    ├─ Conectada al logger del sistema
    ├─ Mensajes descriptivos:
    │  • "Navegando a la página..."
    │  • "Analizando estructura HTML..."
    │  • "Iniciando navegador..."
    │  • "Extrayendo links..."
    │  • "Rankeando resultados..."
    └─ Máximo 100 caracteres para no desbordar
